# OneDoc 简介
OneDoc 是一款轻量级的企业（项目）文档协同管理系统。基于 [MinDoc](https://github.com/mindoc-org/mindoc) 二次开发。

OneDoc 适用于以下应用场景：
+ 项目技术资料管理
+ 企业文档资料管理
+ 互联网博客和在线教程网站

## 更新说明
OneDoc 基于 [MinDoc](https://github.com/mindoc-org/mindoc) 二次开发，更新了以下内容：
- 更新了前端页面样式
- 增加了教程分类的图片
- 增加了文章封面的图片
- 增加了文章阅读次数
